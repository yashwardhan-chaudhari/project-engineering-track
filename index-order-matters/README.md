# Index Order Matters - Investigation

This repository contains the investigation for Challenge #4: Index Order Matters.

## Files Included

- `Changes.md`: A detailed explanation of why the original index was ineffective, an overview of the Left-Most Prefix Rule, and how the corrected index optimized the query performance.
- `experiment.sql`: The SQL commands used to analyze the execution plans before and after fixing the index.

## How to Test
You can run the SQL script `experiment.sql` in a PostgreSQL database initialized with the provided schema to reproduce the `EXPLAIN ANALYZE` observations.
